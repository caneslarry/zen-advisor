CitiesDirectory is Full Application solution(iOS & Android) with Backend Management System.

Used Technology:
- Swift
- Alamofire 
- CoreLocation 
- xCode7.1 and above
- Apple Map 
- Push Notification 
- CodeIgniter 
- Bootstrap 
- RESTFUL API
- Google Chart API
- JSON
